About This Website

About

This website serves as a portfolio-style presentation for multimedia and authoring outputs. It brings together creative process, software knowledge, and project documentation in one space.

Purpose

A home for academic and creative work.

The site is designed to present projects clearly and professionally. It highlights not only the final outputs, but also the ideas, methods, and tools used to create them.

Theme

Pastel visuals with project-specific moods.

The website uses a soft pastel homepage and supporting pages for a playful creative identity, while individual projects like the solar system and rotoscoping pages are allowed to have their own distinct visual themes.

What This Site Communicates

Skills expressed through design and presentation

Creativity

Visual storytelling across media

Each page is designed to reflect artistic thinking, not just technical completion.

Technical Skill

Authoring with digital tools

The project pages show how software and technique support multimedia production.

Presentation

Clear communication of process

The layout helps viewers understand what was made, how it was made, and why it matters.
