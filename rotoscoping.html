Project Page

Rotoscoping

Rotoscoping focuses on tracing movement frame by frame to create animation that feels fluid, expressive, and closely connected to real-life performance.

Overview

Transforming reference footage into stylized motion.

This project studies gesture, timing, and silhouette. By carefully tracing selected frames, the final animation keeps the realism of the original action while introducing a more artistic and authored look.

Process

How the animation is built

  • Select reference footage with clear movement and readable poses.
  • Break the action into key frames and in-between transitions.
  • Trace important contours while simplifying distracting details.
  • Refine line quality, rhythm, and visual consistency.

Creative Focus

What makes the project effective

Motion

Captures believable movement

The animation keeps the energy of the source performance while controlling pacing and emphasis.

Style

Balances realism and interpretation

Lines, shapes, and simplification choices create a more intentional visual identity.

Craft

Requires patience and consistency

Each frame contributes to smoothness, making accuracy and discipline central to the result.
