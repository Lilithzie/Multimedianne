Tools and Software

Apps Used

These projects rely on digital tools for drawing, editing, animation, compositing, 3D modeling, and presentation. Each one supports a different part of the creative workflow.

Animation

Adobe After Effects

Useful for motion graphics, compositing, sequencing frames, and refining animated output.

Editing

Adobe Premiere Pro

Supports video cutting, arrangement, timing adjustments, and polished export preparation.

Illustration

Adobe Photoshop

Helpful for frame cleanup, digital painting, texture work, and visual enhancement.

3D Authoring

Blender

Used to model the solar system scene, manage materials, and create cinematic renders.

Presentation

Canva or PowerPoint

Organizes project visuals and explanations into a clean format for reporting and sharing.

Support

Reference Libraries

Images, videos, and visual studies help guide color, composition, and content accuracy.
